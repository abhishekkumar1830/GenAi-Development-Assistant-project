# genai-developer-assistent
The AI Developer Assistant is a GenAI-powered platform designed to help developers and students enhance their coding skills and career preparation. It provides features like code explanation, debugging, and optimization, enabling users to understand complex programs, identify errors, and improve code efficiency. The platform also supports interview preprations and also analyze the resume according to role and give the resume score...A complete coding platform where no need to write any prompts best for those student who are just started their carrier in technical field does not know how to use chatgpt and how to write proper prompt. this AI Developer Assistant has very simple ui and features so any type of user can use very easily...

## ⚙️ Setup
1. Clone repo
2. Add .env file
3. Run app.py 

## ✨ Features
- 📘 Code Explanation (step-by-step understanding)
- 🐞 Code Debugging (error detection & fixes)
- ⚡ Code Optimization (improves performance)
- 🎯 Interview Question Generator
- 🎤 AI Interview Answer Assistant
- 📄 Resume Analyzer (ATS score + suggestions)
- 🤖 No prompt knowledge required (beginner friendly UI)
